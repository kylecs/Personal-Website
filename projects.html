Projects

These are projects that I have created in my free time. I create these either to learn about a specific field in programming, to become more familiar with a language, or just because I find it fun.

x86 Bare Metal Shell (source code)

This project is a program that runs directly on x86 hardware to provide a low level shell. Interrupts are used to respond to the keyboard and the screen is controlled via memory mapped io.

I created this project with C and x86 assembly. The code is compiled, assembled, and linked without any external libraries. The linked object is then either run in a QEMU emulator or built into an ISO for use on real hardware.


Rust HTTP Server (source code)

This is an HTTP/1.1 web server written in Rust. It complies with the RFC specification and has full support for serving static websites.

I created this project to become more familiar with HTTP and also to get some experience using Rust.


Earth Image Timelapse (source code)

This is a script that uses NASA's Earth Image API to get all public satellite images from given coordinates over a given time period. Images are then filtered for clouds and the remaining images are converted into a timelapse gif using ImageMagick programmatically.


Krypt (source code)

This is an Android App that allows you to send Bitcoin from Coinbase to another person just by touching your phones together. This is accomplished using the Coinbase API and NFC in Android phones.

This is not one of my standard side projects. My team and I created this at the HackPSU Fall 2017 Hackathon. That means that we had 24 hours straight to build the most useful and interesting piece of software we could. Krypt won 3rd place out of 64 teams that submitted projects.

My role in this project was interfacing with the Coinbase API, and integrating that functionality with NFC and our app's interface.
